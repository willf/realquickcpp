Learning More

This real quick guide is really too quick. Here are some additional references which you'll find useful.

Of the making of C++ books there is no end

Here are a few books that you find useful, with links to Amazon.com:

  • Essential C++ by Stanley B. Lippman. This is an excellent tutorial, somewhat in the same spirit as the Real Quick tutorial, but much more complete. (By the way, I would not recommend its companion volume, Accelerated C++, which seems spotty to me in its coverage).
  • The C++ Programming Language (3rd Edition) by Bjarne Stroustrup. Stroustrup is the creator of C++, and this book is a very complete, generally readable reference book that also makes a reasonable tutorial introduction if you are already familiar with programming.
  • Computing Concepts with C++ Essentials, 3rd Edition by Cay Horstmann. I got sent this book by mistake (we used Horstmann's Java version of this book for a intro to programming course I helped teach), but I'm glad I got it. If you're new to programming, and need to start by using C++, this is a good book to do it with.

Net Knowledge

Here are some on-line resources:

  • Standard Template Library Programmer's Guide. This is an essential on-line guide and reference to the Standard Template Library. Since the Standard Template Library is what makes C++ usable, you'll want to review this site.
  • The cplusplus.com tutorial. A pretty good on-line tutorial, which doesn't assume you already know C. They also have a relatively complete reference guide to Standard C++ Library (i.e., the parts not covered in the STL).
  • Java to C++ Transition Tutorial at Brown University.

Code read

You might find the code from this tutorial useful. Feel free to use it at your own risk.

  • countch.cpp (text version). Counts and desplays character frequencies.
  • UnigramTextClassifier.h (text version). UnigramTextClassifier.cpp (text version). Text classifer class based on unigram frequencies. Doxygen created documentation.
  • ilearn.cpp (text version). hamspam.cpp (text version). Makefile (text version). Learner and classifier for ham vs. spam, based on UnigramTextClassifier.

Toolin' around
